## 手机预览功能
<mobile src="https://caniuse.com/" type="ipx" iswechat="true" scale="90"/>

## 编写方式
使用 mobile 标签
1. src 表示类型地址
2. type 表示类型 目前只有 ipx与空 
3. iswechat 表示是否开启小程序的左上角图标
4. scale 表示窗口缩放比例

<br/>

```html
<mobile src="https://caniuse.com/" type="ipx" iswechat="true" scale="90"/>
```
